Apresentação JavaScript

1

O que é JavaScript?

•É uma linguagem de programação de uso geral para desenvolvimento web e de software. Focado principalmente na parte de “Front-end” (parte visual da aplicação), é utilizado junto a outras duas linguagens, HTML e CSS.


•Ainda assim pode ser utilizada com muita versatilidade tendo em vista que pode ser vista em back-ends para processamentos de informações num banco de dados, por exemplo.


•Teve seu surgimento na década de 90’, se embalando no estouro que estava sendo o Java na época (apesar das duas linguagens não serem a mesma coisa), sendo criada através de uma jogada de marketing.

3

Bibliotecas e Frameworks

As bibliotecas são um conjunto de funções prontas com objetivo de facilitar e acelerar o desenvolvimento, mantendo o controle do fluxo.

Exemplo: jQuery, React e Moment.


Os frameworks são estruturas mais completas que modulam o processo de desenvolvimento.

Exemplos: Angular, Bootstrap e Vue.

4

A importância do JavaScript

•Primeiramente, o JavaScript é uma linguagem de programação que permite a implementação de elementos complexos e dinâmicos em páginas da internet.


•Por meio dele, é possível manipular, calcular e validar informações, além de atualizar e aperfeiçoar códigos feitos em HTML e CSS.


•Além disso, o JavaScript existe há mais de 20 anos e está há mais de oito anos sendo a linguagem de programação mais comum entre pessoas desenvolvedoras do mundo todo.

Por que aprender?

1 - Linguagem muito popular

Bastante recomendada para quem deseja iniciar no mundo da programação, pois é fácil de entender quando comparada com outras como o Java, por exemplo.

2 - Sintaxe simples

Uma das características que faz o JavaScript ser bastante utilizado no mercado é a facilidade para entender e utilizar, pois possui uma sintaxe muito amigável.

3 - Usado no Back-End e no Front-End

Uma das grandes vantagens de utilizar o JavaScript é que você usa a mesma linguagem tanto no Front-End quanto no Back-End.

4 - Versatilidade

•Desenvolvimento Web
•Desenvolvimento Mobile
•Desenvolvimento de Jogos

7

Empresas que utilizam JavaScript

Hoje, estima-se que a maior parte da web tenha sido construída em Javascript. Alguns sites que utilizam a tecnologia:


  • Google: Boa parte das interações do Google é em Javascript. Um exemplo disso é a ferramenta de autocompletar da barra de pesquisa; outro é a rolagem infinita da pesquisa de imagens;

  • Twitter: A rolagem infinita da timeline é em JS, bem como a divisão do site em colunas.

  • Netflix: A reprodução automática de partes de um filme/série quando posicionamos o mouse sobre a miniatura é JS.

8

Mercado de trabalho

•O JavaScript é uma das linguagens que mais ganhou destaque nos últimos anos.

•A popularização de seu uso em aplicações back-end e o aumento na demanda por aplicações SPA (Single Page Application) através de frameworks como o Angular e o React fez com que o JavaScript desse um salto de popularidade e em soluções computacionais de variados níveis de complexidade e se tornou tão popular.
